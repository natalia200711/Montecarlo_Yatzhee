# Simulación del Juego Yahtzee con Método de Montecarlo

## Descripción del proyecto

Este proyecto consiste en el desarrollo de una simulación del juego clásico Yahtzee utilizando el lenguaje de programación Python y el método de Montecarlo.

El programa permite ejecutar múltiples partidas automáticas entre dos jugadores virtuales, utilizando lanzamientos aleatorios de dados para analizar el comportamiento probabilístico del juego y obtener estadísticas sobre victorias y empates.

La simulación se basa en la generación de números aleatorios con distribución uniforme, donde cada cara del dado tiene la misma probabilidad de aparecer.

---

## Objetivo del proyecto

Desarrollar una simulación computacional del juego Yahtzee aplicando el método de Montecarlo para analizar resultados aleatorios y estudiar el comportamiento estadístico obtenido a partir de múltiples ejecuciones del sistema.

---

## Características del programa

El sistema desarrollado permite:

- Simular lanzamientos de cinco dados.
- Ejecutar hasta tres lanzamientos por turno.
- Evaluar automáticamente las combinaciones obtenidas.
- Asignar puntajes según las reglas implementadas.
- Simular partidas entre dos jugadores.
- Realizar múltiples simulaciones utilizando el método de Montecarlo.
- Mostrar estadísticas finales de victorias y empates.

## Funcionamiento general

El programa realiza simulaciones automáticas del juego Yahtzee.

En cada partida:

1. Cada jugador lanza cinco dados.
2. El sistema permite hasta tres lanzamientos.
3. Los dados con valores bajos pueden volver a lanzarse automáticamente.
4. Se evalúa la combinación obtenida.
5. Se asigna un puntaje.
6. Se comparan los resultados de ambos jugadores.
7. Se registra el ganador o empate.

El proceso se repite múltiples veces para obtener resultados estadísticos.

---

## Método de Montecarlo

El método de Montecarlo consiste en realizar simulaciones repetidas utilizando valores aleatorios para aproximar el comportamiento de un sistema probabilístico.

En este proyecto, el método se aplica mediante la ejecución de múltiples partidas aleatorias de Yahtzee, permitiendo analizar frecuencias de victoria, empates y estabilidad de los resultados obtenidos.
